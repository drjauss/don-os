# Don Os Static web app project

Static web site to share and publish reviews and info about Oswaldo Diaz Diaz and his work.

## Frameworks and dependencies

- ReactJS
- Pure.css grid and responsive layout
- Webpack
- SASS
- TypeScript
- Font Awesome
